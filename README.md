Due to the significant training and inference time for the various configurations, specifically the Scratch-B models, which require a larger compute budget of the specified epochs, and the high computational cost of the Hessian eigenspectrum analysis, a single executable notebook is not feasible to run within a standard environment. Instead, we have provided a dedicated README for each research line:
* CNN methods
* ViT structured pruning
* Loss Landscape geometry

These detail the exact commands to run and specify the location of our existing results (CSV files) and visualizations, such as the 2D loss surfaces and linear mode connectivity plots.
